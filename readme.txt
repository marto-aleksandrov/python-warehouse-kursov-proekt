Система за складова наличност (Warehouse / Inventory)
=====================================================

Малък проект на Python, който моделира склад със стоки. Демонстрира
обектно-ориентирано програмиране (клас с конструктор и няколко метода),
заедно с цикли, условни изрази и сортиране.

СТРУКТУРА НА ПРОЕКТА
-------------------
models.py          - Дефинира класовете Product и Warehouse.
main.py            - Изгражда примерен склад и изпълнява демонстрацията
                     (сортиране, откриване на ниски наличности, зареждане,
                     обща стойност).
requirements.txt   - Външни библиотеки, необходими за стартиране на проекта.
readme.txt         - Този файл.

КЛАСОВЕ (models.py)
-------------------
Product (Продукт)
  __init__(name, quantity, price, category)  - конструктор (използва self)
  total_value()                              - количество * цена
  is_low_stock(threshold=5)                  - True, ако наличността е ниска
  restock(amount)                            - добавя бройки към наличността

Warehouse (Склад)
  __init__(name)
  add_product(product)
  total_inventory_value()                    - сума на стойностите на всички стоки
  find_low_stock(threshold=5)                - стоки, които трябва да се поръчат
  sort_by_quantity(descending=False)         - сортира стоките по количество
  sort_by_value(descending=True)             - сортира стоките по обща стойност

ИЗИСКВАНИЯ
----------
Python 3.8 или по-нов.
Външни библиотеки (виж requirements.txt):
  - tabulate   (използва се за извеждане на наличността като форматирани таблици)

ИНСТАЛАЦИЯ
----------
1. (По избор) Създайте и активирайте виртуална среда:
     python -m venv venv
     source venv/bin/activate         (Windows: venv\Scripts\activate)

2. Инсталирайте необходимите библиотеки:
     pip install -r requirements.txt

КАК ДА СТАРТИРАТЕ
----------------
   python main.py

Програмата ще:
  1. Изгради склад с примерни стоки.
  2. Изведе стоките, сортирани по количество (възходящо).
  3. Изведе стоките, сортирани по обща стойност (низходящо).
  4. Изброи стоките с ниска наличност и ги отбележи за поръчка.
  5. Зареди стоките с ниска наличност и изведе обновената наличност.
  6. Изведе общата стойност на целия склад.
